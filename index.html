Release templates for ZEF Global applications.

Below you will find a collection of release pages we have made for various ZEF Global applications. Please feel free to use these as a templates for your own custom release pages. Just remember to change the application ID in the templates to match your application ID! Also please notice that the graphic material on the release page might be copyrighted so you should use your own graphics!

You can find more information about the templates from the GitHub page which you can access by clicking the 'View on GitHub' button at the top of this page. If you need help with the templates please don't hesitate to contact us by email: support@zef.fi.
